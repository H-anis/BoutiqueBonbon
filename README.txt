Site de Bonbon

Ceci est un site marchand de bonbons:

	-Afin d'ajouter un produit, veuillez cliquer sur la rubrique modifications
	-Afin de supprimer un produit, veuillez cliquer sur le bouton Supprimer sur le produit correspondant.
	-Afin de modifier un produit, veuillez cliquer sur le bouton Modifier sur le produit correspondant.
